## 📌 Deskripsi Proyek
Proyek ini membandingkan efisiensi algoritma pencarian data antara **Binary Search Tree (BST)** dengan **Linear Search** (pencarian sekuensial) menggunakan sekumpulan data uji acak.

Hasil pengujian membuktikan bahwa pencarian menggunakan BST jauh lebih hemat perbandingan (*comparison count*) dengan kompleksitas rata-rata $O(\log n)$ dibandingkan dengan pencarian linear yang memiliki kompleksitas $O(n)$, dengan catatan pohon dalam kondisi relatif seimbang.
